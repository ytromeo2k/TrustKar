# TRUSTKAR — Naya Start (bilkul saaf)

Purane Git / GitHub errors khatam. **Sirf yeh steps follow karo — order change mat karo.**

---

## PART 1 — GitHub par purana repo hatao (2 minute)

1. Browser: [github.com/ABBASCAAN/trustkar](https://github.com/ABBASCAAN/trustkar)
2. **Settings** (repo ke andar, upar right)
3. Neeche scroll → **Danger Zone**
4. **Delete this repository** → naam likho `ABBASCAAN/trustkar` → confirm

5. Phir **+** → **New repository**
   - Name: `trustkar`
   - **Public**
   - ❌ README mat tick karo
   - ❌ .gitignore mat tick karo
   - **Create repository**

Ab GitHub **bilkul khali** hai. Koi conflict nahi hoga.

---

## PART 2 — PC par purana Git hatao (1 minute)

1. Folder kholo: `C:\Users\ZC\Projects\trustkar`
2. **`.git`** folder dikhe to **Delete** karo (hidden ho to View → Hidden items ON)
   - Ya **`fresh-start.bat`** chalao — woh khud karega

---

## PART 3 — Ek click upload (5 minute)

1. **`fresh-start.bat`** par **double-click**
2. GitHub login (browser)
3. Jab likhe **SUCCESS** — khatam

---

## PART 4 — Vercel (ek bar)

**Agar pehle project hai:**
- Vercel → project → **Settings** → **Git** → Disconnect
- **Add New Project** → naya `trustkar` repo → **Deploy**

**Agar nahi hai:**
- [vercel.com/new](https://vercel.com/new) → `trustkar` → **Deploy**

---

## PART 5 — Firebase (ek bar, 3 minute)

[console.firebase.google.com](https://console.firebase.google.com) → `romeo-escrow`

1. **Authentication** → Email/Password → Enable
2. **Firestore** → Create database
3. **Rules** → `firestore.rules` copy paste → Publish

---

## Roz ka kaam (ab se sirf yeh)

1. Cursor se code badlo
2. **`upload-github.bat`** double-click
3. 2 min wait → site live

**Git Bash mat kholo.**
